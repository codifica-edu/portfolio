Linguagens

HTML

HTML (HyperText Markup Language) é a base de qualquer site, definindo a estrutura e o conteúdo das páginas da web. Com minhas habilidades em HTML, já criei bibliotecas virtuais, blogs e sites pessoais.

CSS

CSS (Cascading Style Sheets) é a linguagem que define o estilo e o layout de páginas web. Sei usar CSS para manipular cores, fontes, imagens, criar animações e mais, dando vida e personalidade aos meus projetos.

JavaScript

JavaScript é a linguagem que traz interatividade às páginas web. Sei usá-lo para alterar o DOM, realizar operações matemáticas, enviar formulários, alimentar tabelas etc. Tornando meus projetos mais dinâmicos e funcionais.
